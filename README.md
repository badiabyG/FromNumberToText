Application pour convertir un nombre écrit avec des chiffres en nombres écrit avec des lettres :

Pour lancer l'application il suffit d'executer le fichier Application/main.java

Vous aurez alors plusieurs choses affichés dans une fenêtre :
- Un champ de texte
- Un bouton intitulé "Convertir le nombre écrit dans le champ"
- Un bouton intitulé "Convertir un nombre au hasard"


Si vous n'écrivez rien, que vous tapez autre chose qu'un nombre/chiffre ou que votre nombre n'est pas inclu dans la page des entiers de Long de Java et que vous appuyez sur le bouton intitulé "Convertir le nombre écrit dans le champ", une erreur sera affichée.

Si vous taper un nombre valide, il y aura un texte dans l'espace entre le champ et les boutons qui vous montrera le conversion.

Si vous appuyez sur le bouton intitulé "Convertir un nombre au hasard" à n'importe quel moment, il y aura un texte dans l'espace entre le champ et les boutons qui vous montrera le conversion d'un nombre choisi au hasard.

Pour fermer l'application, il suffit d'arreter l'execution ou appuer sur la croix en haut à drote de la fenêtre.
